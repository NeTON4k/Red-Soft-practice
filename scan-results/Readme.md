# Scan Results

В данном каталоге собраны материалы, относящиеся к использованию инструментов анализа защищенности.

## Содержимое

- `nmap/` — исследование сетевых сервисов;
- `nikto/` — анализ веб-сервиса;
- `searchsploit/` — поиск известных эксплойтов;
- `hydra/` — проверка устойчивости механизмов аутентификации;
- `responder/` — анализ сетевого взаимодействия;
- `lynis/` — аудит безопасности операционной системы;
- `openscap/` — проверка соответствия рекомендациям по безопасности.
