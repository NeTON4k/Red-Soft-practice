# Hydra

## Назначение

Hydra использовалась для демонстрации возможности подбора учетных данных в лабораторной среде.

## Выполненные задачи

- проверка устойчивости службы аутентификации;
- моделирование атаки методом перебора паролей.

## Использование в отчете

Результаты использованы при рассмотрении вопросов безопасности аутентификации.
