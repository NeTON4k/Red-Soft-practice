# Lynis

## Назначение

Lynis использовался для локального аудита безопасности операционной системы РЕД ОС.

## Выполненные задачи

- анализ конфигурации системы;
- поиск потенциально небезопасных настроек;
- формирование рекомендаций по повышению защищенности.

## Использование в отчете

Результаты аудита использованы при подготовке рекомендаций по усилению безопасности.
