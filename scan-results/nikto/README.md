# Nikto

## Назначение

Nikto применялся для автоматизированной проверки веб-сервиса на наличие известных проблем безопасности и ошибок конфигурации.

## Выполненные задачи

- анализ HTTP-сервиса;
- поиск небезопасных настроек;
- выявление потенциально уязвимых компонентов.

## Использование в отчете

Полученные результаты использовались при анализе безопасности веб-службы.
