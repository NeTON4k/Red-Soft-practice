# Nmap

## Назначение

Nmap использовался для исследования сетевой доступности виртуальной машины РЕД ОС, обнаружения открытых портов и определения версий работающих сетевых служб.

## Выполненные задачи

- обнаружение активных сервисов;
- определение открытых TCP-портов;
- идентификация версий служб;
- сбор информации для дальнейшего анализа потенциальных уязвимостей.

## Использование в отчете

Результаты работы Nmap использованы при описании этапа разведки целевой системы.
