# OpenSCAP

## Назначение

OpenSCAP использовался для автоматизированной проверки соответствия системы рекомендациям по информационной безопасности.

## Выполненные задачи

- аудит конфигурации;
- проверка соответствия политикам безопасности;
- анализ состояния защищенности системы.

## Использование в отчете

Результаты проверки использованы при оценке общего уровня защищенности операционной системы.
