# Responder

## Назначение

Responder применялся для анализа сетевого взаимодействия и демонстрации атак на протоколы разрешения имен в локальной сети.

## Выполненные задачи

- анализ сетевых запросов;
- исследование механизмов LLMNR и NetBIOS;
- демонстрация возможности перехвата учетных данных.

## Использование в отчете

Полученные данные использованы при анализе безопасности сетевой инфраструктуры.
