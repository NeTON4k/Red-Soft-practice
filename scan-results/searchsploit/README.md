# Searchsploit

## Назначение

Searchsploit использовался для поиска публично известных эксплойтов по обнаруженным версиям программного обеспечения.

## Выполненные задачи

- поиск соответствующих CVE;
- анализ доступных эксплойтов;
- оценка возможности эксплуатации обнаруженных сервисов.

## Использование в отчете

Результаты поиска были использованы при анализе потенциальных уязвимостей.
